## Requirements
- gym
- gym[atari]
- tensorflow
- keras
- h5py
- numpy
- pickle (optional)


## How to use
Use the argument.py file to modify the arguments to your liking.

Then, run the main.py file by entering the command, `python main.py` into the terminal.
